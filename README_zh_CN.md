## 关于 NoBuildPlus

## 简介
NoBuildPlus提供了多种不同的规则来保护整个世界, 在简单与自定义的配置下，你能够轻松地管理服务器中的世界。
- 简易配置文件管理整个世界
- 能够配置规则所管理的物品、生物及方块等
- 可以为世界添加默认设置
- 无视权限及消息提示都能在配置文件中自定义
- 方便实用的菜单

## 版本支持
- 1.8+ (Spigot, Paper, Purpur)
